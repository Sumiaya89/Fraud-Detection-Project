# Fraud Detection Analysis Using Data Analytics

## Project Overview

This project detects fraudulent transactions using Machine Learning.

## Technologies Used

* Python
* Pandas
* Seaborn
* Matplotlib
* Scikit-Learn

## Dataset Information

* Total Transactions: 10000
* Normal Transactions: 9750
* Fraud Transactions: 250

## Model Used

* Random Forest Classifier

## Result

* Accuracy: 99.7%

## Conclusion

The model successfully identified fraudulent transactions with high accuracy.
